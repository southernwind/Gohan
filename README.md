# Gohan
うちの毎日の献立管理用アプリ

* レシピダウンロード機能
    * CookPad
    * つくおき
    * 楽天レシピ
    * WeekCook
* 保存レシピ検索機能
* マルチキャストによるローカルネットワーク内更新通知機能

などなど...
